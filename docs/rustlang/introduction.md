## Introduction

The Azure SDK for Rust is *not officially supported* but still follows similar guidelines as other language SDKs. These guidelines expand on but take precedence over [general Azure SDK guidelines][general-guidelines] to provide SDKs that are similar to other languages' SDKs, but idiomatic for [Rust][rustlang-site] developers (rustaceans).
